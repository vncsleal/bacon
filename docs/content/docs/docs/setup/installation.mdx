## Clone and Install

```sh title="Terminal"
git clone https://github.com/vncsleal/bacon.git my-saas
cd my-saas
pnpm install
```

## Convex Setup

bacon requires Convex for authentication and database functionality. Run the following to set up your Convex deployment:

```sh title="Terminal"
cd packages/database
npx convex dev
```

This will:
1. Create a Convex development deployment
2. Push the database schema
3. Generate TypeScript types
4. Start a hot-reloading dev server

Leave this terminal running.

## Environment Variables

Copy the `.env.example` files and configure your variables:

```sh title="Terminal"
cp .env.example .env.local
```

See the [Environment Variables](/docs/setup/env) guide for detailed configuration.

## Development

Start the development servers:

```sh title="Terminal"
pnpm dev
```

Open the localhost URLs with the relevant ports:

- [http://localhost:3000/](http://localhost:3000/) — The main app.
- [http://localhost:3001/](http://localhost:3001/) — The website.
- [http://localhost:3002/](http://localhost:3002/) — The API.
- [http://localhost:3003/](http://localhost:3003/) — Email preview server.
- [http://localhost:3004/](http://localhost:3004/) — The docs.
