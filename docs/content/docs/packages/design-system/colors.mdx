Colors in bacon are organized around a central CSS file that defines the application's design tokens, ensuring consistency across the application.

## CSS Variables

We use CSS custom properties for colors, and Tailwind CSS is configured to consume them.

## Changing the Primary Color

To change the primary color, update the CSS variables in the color file. The system is based on HSL values which allows for easy customization.

1. Open the color configuration file
2. Update the `--primary` and `--primary-foreground` values
3. The rest of the application will automatically reflect the change
