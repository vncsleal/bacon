The Next.js configuration in bacon uses packages to keep your Next.js configuration modular and easier to manage.

## Image Optimization

The package configures Next.js image optimization to support AVIF and WebP formats.

## Bundle Analysis

To analyze your production bundle size, run:

```sh
pnpm analyze
```

This will generate an interactive treemap visualization of your bundle.
