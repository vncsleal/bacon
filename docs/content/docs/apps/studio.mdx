<Info>
  The Studio app has been removed from bacon. The Convex dashboard provides a built-in data browser for viewing and editing your database.
</Info>

## Convex Dashboard

Visit your [Convex dashboard](https://dashboard.convex.dev) to:

- Browse and edit database tables
- View function logs and metrics
- Configure environment variables
- Manage deployments

Alternatively, run `npx convex dashboard` from the `packages/database` directory to open the dashboard locally.
